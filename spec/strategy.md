# EMA 多通道 + MACD 动能趋势策略 (5M)

## 概述
本策略基于多周期指数移动平均线 (EMA) 与 MACD 动能指标结合，通过趋势过滤与动能突破信号识别短线交易机会。  
适用于高波动币种（如 SOL、BTC、ETH），在 5 分钟周期下进行短线趋势跟随交易。

---

## 指标设定

| 指标 | 参数 | 用途 |
|------|-------|------|
| EMA8  | 周期 8  | 快速趋势识别 |
| EMA30 | 周期 30 | 短期趋势过滤 |
| EMA55 | 周期 55 | 中期支撑/阻力 |
| EMA144 | 周期 144 | 长期趋势方向 |
| EMA169 | 周期 169 | 趋势防线 |
| MACD | 快线=12, 慢线=26, 信号线=9 | 动能确认 |

---

## 趋势判定

### 多头趋势条件
- EMA8 > EMA30 > EMA55 > EMA144 > EMA169  
- K线收盘价位于 EMA8 与 EMA30 之上  
- MACD 柱线 > 0  

→ 市场为上升趋势，仅考虑做多信号。

### 空头趋势条件
- EMA8 < EMA30 < EMA55 < EMA144 < EMA169  
- K线收盘价位于 EMA8 与 EMA30 之下  
- MACD 柱线 < 0  

→ 市场为下降趋势，仅考虑做空信号。

---

## 入场规则

### 做多信号
- EMA8 上穿 EMA30；  
- MACD 快线向上突破慢线；  
- 成交量略放大；  
- 当前K线收盘价高于 EMA8。

**执行操作：** 开多仓。

### 做空信号
- EMA8 下穿 EMA30；  
- MACD 快线向下穿越慢线；  
- 成交量略放大；  
- 当前K线收盘价低于 EMA8。

**执行操作：** 开空仓。

---

## 出场规则

### 止盈逻辑
- 价格相对 EMA8 偏离 ≥ 1.5% 时，分批止盈；  
- 或 MACD 柱线开始缩短（动能减弱）时，全平仓。

### 止损逻辑
- 收盘价跌破（或上破）EMA30；
- 或固定止损范围为 0.8%～1.2%。

---

## 持仓管理
- 若价格始终运行于 EMA144、169 之上（大趋势强势），在价格回踩 EMA55 并反弹时可加仓；
- 若价格跌破 EMA144/169，停止加仓或清仓等待新趋势。

---

## 优化与过滤条件
- 可结合 RSI(14)：  
  - RSI > 50 → 多头信号有效  
  - RSI < 50 → 空头信号有效  
- 多周期确认：  
  - 建议在 15M 与 1H 周期中趋势方向一致时进场。
- 若 MACD 与价格背离 → 提前减仓或平仓。

---

## 策略适用范围
- **时间周期：** 5 分钟  
- **适用品种：** SOL、BTC、ETH、其他高波动主流币  
- **市场类型：** 永续合约、现货趋势行情  
- **平台示例：** Backpack、Binance、Bybit、OKX  

---

## TradingView 实现要点 (Pine Script 提示)
- 使用 `ta.ema()` 生成 EMA；
- 使用 `ta.macd()` 获取 MACD 三线；
- 条件触发时绘制买卖箭头；
- 可添加偏离报警函数 `alertcondition()` 实现自动提醒。
